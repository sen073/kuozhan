# 狂人之家书屋 · 方案B 系列聚合书源

站点：https://www.crazyhome2000.com/

把分散的文章块（如 `满座仙魔尽裙臣 162-167`）按书名聚合成一本可读系列。

## 网络导入

```text
https://raw.githubusercontent.com/sen073/kuozhan/main/legado-crazyhome2000/booksource.json
```

导入入口：**我的 → 书源管理**（不是订阅源）

## 怎么用

1. 搜索完整或半书名，例如：`满座仙魔尽裙臣`、`妖尘录`
2. 结果里是**书名**（已聚合），不是碎章节
3. 点进后目录按章节区间排序：`第1-20章`、`第21-28章`…
4. 每一“章”对应网站一篇文章块，正文为该块全文

## 原理

```text
搜索: WP REST /wp-json/wp/v2/posts?search=关键词
  → 标题去尾部章节区间 → 按系列名聚合

目录: 再搜系列名 → 过滤同名系列 → 按起始章排序

正文: /wp-json/wp/v2/posts/{id} → content.rendered
```

## 限制

- 成人向站点，内容请自行甄别
- 标题不规范时可能聚合不全
- 关键词太短会比较吵，尽量用完整书名
- 支持正版与合法使用
